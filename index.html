How to Get a Programming Job

Adam Johnson

✨ Inspiration

  • How to Get Hired as a Django Developer by Will Vincent

🤨 Interviewing

  • First job is the hardest.
  • Interviewing is its own skill.
  • There’s huge demand for developers, but also many chancers.

👀 GitHub Profile

  • Good to have some code on display, and to talk about.
  • ...but don’t stress about doing tons, keeping a streak, etc.
  • Fill in your profile, add README.

👽 Coding Challenges

  • Advent of Code - annual series of challenges, with multiple difficulty levels.
  • Project Euler - mathematical programming challenges.
  • CodinGame - learn programming through a game.
  • Tutorials: Django Tutorial, Django Girls, etc.

⌨️ Blog

  • Good programmers are good writers.
  • Blogging lets you practice writing, and gives interviewers something to look at and ask you about.
  • If you set up a blog and do a minimal update annually, you’re better than 90% of developers.

🤖 Blog Software

  • Set it up free with GitHub Pages, Render Static Sites, etc.
  • A domain is completely optional. <username>.github.io is perfectly fine.
  • Use a static site generator - e.g. Python-based Pelican.

✍️ Blog Content

  • Write about your experiences programming, in your style. Give it a personal touch!
  • Cement your learning: The Feynman Techinque.
  • Don’t worry about being wrong. Don’t be prfectionist.
  • Include an about page, maybe CV.

📚 Blog Formats

  • TIL (Today I Learned) - e.g. Hynek Schlawack, Simon Willison.
  • Weeknotes - e.g. Carlton Gibson, Sage Abdullah.
  • Short posts - e.g. Seth Godin.
  • Massive information dumps - e.g. Haki Benita.

Do whatever works for you.

📋 Community Job Sites

  • Twitter - #TechIsHiring, other searches, ask for help.
  • Hacker News - monthly “Who is Hiring” posts e.g. Dec 2020
  • Job boards - Python, Django Jobs, etc.

💰 Commercial Job Sites

  • Remote OK - remote positions only.
  • Stack Overflow Jobs - high traffic.
  • Freelance sites, e.g. oDesk.
  • TopTal - “elite” freelancing, harder entry test.

🕵️‍♀️ Intelligence Sites

  • levels.fyi - FAANG focused.
  • Glassdoor - general ratings and salaries.

😅 It Never Stops!

  • Brutal truth: best way to get a pay raise is to change jobs.
  • Average length of employment somewhere between 1 and 3 years.
  • Always build your profile - even if writing daily/weekly diary for later CV.
  • Keep checking what’s out there.
